**1️⃣ What is the difference between var, let, and const?**
Ans: **var** is the older way to declare variables in JavaScript. It has function scope and its value can be reassigned. The same variable can also be redeclared in the same scope.
**let** is a modern way to declare variables. It has block scope. Its value can be changed, but the variable cannot be redeclared in the same block.
**const** is used to declare variables whose value should not change. It also has block scope, and the value must be assigned when the variable is created.

**2️⃣ What is the spread operator (...)?**
Ans: The **spread operator** (...) in JavaScript is used to expand elements of an array or object.It lets you copy or merge arrays and objects easily. For example, you can spread values from one array into another array. It is also used to pass multiple values as separate arguments in a function.

**3️⃣ What is the difference between map(), filter(), and forEach()?**
Ans: **map()** creates a new array by applying a function to each element of the original array.
**filter()** creates a new array with only the elements that pass a condition.
**forEach()** runs a function on each element of the array, but does not return a new array.

**4️⃣ What is an arrow function?**
Ans: An **arrow function** is a short and modern way to write a function in JavaScript. It uses the => (arrow) syntax instead of the function keyword. It makes the code shorter and easier to read.

**5️⃣ What are template literals?**
Ans: Template literals are a way to write strings in JavaScript using backticks (`) instead of quotes.They allow you to insert variables inside a string using ${ }.They also make it easy to write multi-line strings.
